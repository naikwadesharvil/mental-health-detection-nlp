# Explainable Transformer-based Detection of Depression and Anxiety from Social Media Text

## 🎯 Project Overview
This research project investigates the effectiveness of transformer-based 
language models (BERT, RoBERTa) for detecting early signs of depression 
and anxiety from social media posts, with an emphasis on model 
explainability using LIME for clinical interpretability.

## 🔬 Research Questions
1. Can fine-tuned transformer models outperform traditional ML approaches 
   in mental health text classification?
2. How do different model architectures (LSTM, BERT, RoBERTa) compare 
   in terms of accuracy, computational cost, and interpretability?
3. Can LIME-based explanations provide clinically meaningful insights 
   into model predictions?

## 📊 Methodology
- **Dataset:** Reddit Mental Health Dataset
- **Models Compared:**
  - Baseline: TF-IDF + Logistic Regression
  - Traditional DL: LSTM with attention
  - Modern: BERT-base fine-tuned
  - Modern: RoBERTa-base fine-tuned
- **Explainability:** LIME (Local Interpretable Model-Agnostic Explanations)
- **Evaluation Metrics:** Accuracy, Precision, Recall, F1-Score, AUROC
